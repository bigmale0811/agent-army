# AgentForge 安裝指南 — 超簡單版

> 這份指南寫給完全不會寫程式的人。
> 只要你會用電腦上網、會複製貼上，就能完成安裝。
> 預計花費時間：**15 分鐘**

---

## 你需要準備什麼

- 一台電腦（Windows 或 Mac 都可以）
- 網路連線
- 一個 Google 帳號（用 Gmail 的那個就行）

---

## 第一步：安裝「程式語言工具」

AgentForge 需要一個叫 Python 的程式語言工具才能運作，就像 Word 需要先安裝 Office 一樣。

### Windows 的做法

1. 打開瀏覽器（Chrome 或 Edge 都行），網址列輸入：
   ```
   python.org
   ```
2. 你會看到一個黃色大按鈕寫著「Download Python」，點它。
3. 下載完成後，**雙擊**打開那個檔案。
4. ⚠️ **非常重要！** 安裝畫面的**最下面**有一行小字：
   ```
   Add python.exe to PATH
   ```
   **務必在旁邊的小方框打勾 ✅**，沒打勾的話後面會出問題。
5. 點「Install Now」，等它跑完。
6. 看到「Setup was successful」就成功，點「Close」關掉。

### Mac 的做法

1. 打開瀏覽器，網址列輸入：
   ```
   python.org
   ```
2. 點「Downloads」，再點「Download Python for macOS」。
3. 下載完雙擊打開，一直點「繼續」和「同意」，直到裝完。
4. 安裝完成，關掉視窗即可。

---

## 第二步：打開「黑色打字視窗」

這個視窗是我們輸入指令的地方，不用害怕，只要照著複製貼上就好。

### Windows 的做法

1. 按鍵盤上的 **Windows 鍵**（左下角那個有窗戶圖案的鍵）。
2. 用鍵盤輸入：`cmd`
3. 看到「命令提示字元」出現，**按 Enter** 打開它。
4. 你會看到一個黑色視窗，裡面有一行閃動的游標，這就對了。

### Mac 的做法

1. 同時按 **Command 鍵（⌘）+ 空白鍵**。
2. 輸入：`Terminal`
3. 看到「終端機」出現，按 Enter 打開。
4. 你會看到一個白色或黑色視窗，裡面有閃動的游標。

---

## 第三步：安裝 AgentForge

在剛才打開的黑色視窗裡：

1. 把以下這行文字**複製起來**，貼到黑色視窗裡，然後按 **Enter**：
   ```
   pip install agentforge
   ```
2. 視窗會開始跑很多文字，這是正常的，等它自己跑完（大約 1～2 分鐘）。
3. 最後看到：
   ```
   Successfully installed agentforge
   ```
   就代表安裝成功了！

> 如果看到紅色文字，請跳到最後的「遇到問題怎麼辦」章節。

---

## 第四步：執行安裝精靈

1. 在同一個黑色視窗裡，複製貼上以下指令，按 **Enter**：
   ```
   agentforge setup
   ```
2. 精靈會用中文一步步問你問題，你只要依照指示回答就好。

3. 精靈會問你要用哪個「AI 服務」，請選數字：
   - 輸入 `1`，按 Enter：選 **Google Gemini**（有免費方案，推薦新手）
   - 輸入 `2`，按 Enter：選 **Claude**（如果你有 Anthropic 訂閱）
   - 輸入 `3`，按 Enter：選 **OpenAI**（如果你有 ChatGPT Plus 帳號）

4. 精靈接著會要求你輸入「通行證」（就像大樓的門禁卡，讓 AgentForge 能夠使用 AI 服務）。

---

## 第五步：取得你的「通行證」

### 如果你選了 Google Gemini（免費）

1. 打開瀏覽器，網址列輸入：
   ```
   aistudio.google.com/apikey
   ```
2. 用你的 Google 帳號登入。
3. 點「Create API key」（建立通行證）。
4. 你會看到一串英文字母和數字，那就是你的通行證，點旁邊的「複製」按鈕複製它。
5. 回到黑色視窗，把通行證貼上去，按 **Enter**。

### 如果你選了 Claude

1. 打開瀏覽器，網址列輸入：
   ```
   console.anthropic.com
   ```
2. 登入帳號後，點左側「API Keys」。
3. 點「Create Key」，複製那串通行證。
4. 回到黑色視窗，貼上，按 **Enter**。

---

## 第六步：試跑你的第一個 AI 助手

1. 在黑色視窗輸入以下指令，按 **Enter**：
   ```
   agentforge run example
   ```
2. 你會看到 AI 開始工作，螢幕上出現一些文字。
3. 等它跑完，最後看到「Done」或「完成」，就大功告成！

---

## 遇到問題怎麼辦

**問題一：黑色視窗出現紅色文字**

這通常是因為網路暫時不穩。試試這樣做：
1. 等 30 秒。
2. 重新輸入一次剛才的指令，再按 Enter。
3. 如果還是不行，重新啟動電腦，再試一次。

---

**問題二：輸入 `agentforge` 後出現「找不到指令」**

這是因為第一步的安裝沒有打勾「Add python.exe to PATH」。
解決方法：
1. 重新按照「第一步」再裝一次 Python。
2. **這次一定要記得打勾** ✅ 那個選項。
3. 裝完重新開啟黑色視窗，再試一次。

---

**問題三：通行證貼上去後說「無效」或「認證失敗」**

1. 確認你複製的時候有複製完整（通行證通常很長，不能只複製一部分）。
2. 確認你用的是正確的 AI 服務帳號（選了 Gemini 就要用 Google 通行證）。
3. 確認帳號有開通 API 功能（部分服務需要先綁定信用卡才能使用）。

---

**問題四：Mac 上說「權限不足」**

在指令前面加上 `sudo `，例如：
```
sudo pip install agentforge
```
它會要求你輸入你的 Mac 登入密碼（輸入時螢幕不會顯示任何字，這是正常的），打完按 Enter。
