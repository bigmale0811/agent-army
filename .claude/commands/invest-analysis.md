# Investment Analysis

Use data-analyst agent to execute:

1. Confirm target (Taiwan stock code / crypto symbol)
2. Scrape historical data (at least 120 days)
3. Calculate technical indicators:
   - Moving Averages (5MA, 10MA, 20MA, 60MA)
   - RSI (14-day)
   - MACD
   - KD Stochastic
   - Bollinger Bands
4. Determine current trend, support/resistance levels
5. Generate analysis report with charts
